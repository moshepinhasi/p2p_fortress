# p2p_fortress
Serverless P2P encrypted file transfer — AES-256-GCM, Argon2id, ECDH X25519, Tor support &amp; certificate pinning. Built with Python &amp; CustomTkinter.


*******************************
Sometimes the file ends up without an extension. The receiving party simply needs to rename the file and add an extension of the original file type.
